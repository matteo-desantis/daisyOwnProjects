# Project Library Documentation

This folder contains project-level class definitions used across custom daisy seed based projects.

## Purpose

- Centralized location for shared class definitions
- Reusable components for Project with different target boards
- Project-wide utility classes and interfaces

## Structure

Classes in this folder should be:
- Well-documented
- Reusable across different modules
- Hardware-independent where possible

## Usage

Include these class definitions in your module projects by referencing this folder.
